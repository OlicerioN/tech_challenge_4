## Tech Challenge – Fase 4  
# Sistema Preditivo e Painel Analítico de Obesidade

Este projeto tem como objetivo **analisar e prever níveis de obesidade**, combinando modelagem preditiva com IA e análise exploratória de dados por meio de um painel analítico interativo.  
A solução foi desenvolvida para apoiar análises populacionais, estudos de saúde e tomada de decisão baseada em dados.


## Objetivos do Projeto

- Desenvolver um modelo de machine learning capaz de predizer o nível de obesidade de um indivíduo
- Criar um aplicativo interativo para predição individual
- Construir um painel analítico com insights relevantes sobre obesidade, hábitos alimentares, atividade física e perfil populacional


## Funcionalidades

### Preditor de Nível de Obesidade (Streamlit)
- Interface interativa e intuitiva
- Entrada de dados clínicos e comportamentais do indivíduo
- Cálculo automático do IMC
- Tabela explicativa de cada item do questionário
- Aplicação totalmente em português


### Painel Analítico (Power BI)
- Distribuição dos níveis de obesidade na população
- Análises por:
  - Idade
  - Gênero
  - Consumo de alimentos calóricos
  - Prática de atividade física
  - Hidratação diária
  - Histórico familiar de obesidade
- Indicadores percentuais (KPIs), como:
  - Percentual de obesos que consomem alimentos calóricos
  - Relação entre sedentarismo e obesidade
- Gráficos interativos para apoio à análise e comunicação de dados


## Tecnologias Utilizadas

- Python
- Pandas / NumPy
- Scikit-learn
- Streamlit
- Power BI
- Git & GitHub
